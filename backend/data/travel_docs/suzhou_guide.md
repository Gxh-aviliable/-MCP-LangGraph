# 苏州旅游攻略

## 概述

苏州，古称姑苏、平江，是江苏省下辖的地级市，位于江苏省东南部，东临上海，南接嘉兴，西抱太湖，北依长江。苏州是中国首批24座国家历史文化名城之一，有"人间天堂"的美誉。

## 必游景点

### 1. 拙政园
- **简介**: 中国四大名园之一，苏州园林的代表作品
- **门票**: 70元（淡季50元）
- **游玩时间**: 2-3小时
- **特色**: 以水为中心，山水萦绕，厅榭精美，花木繁茂

### 2. 留园
- **简介**: 中国四大名园之一，以建筑艺术著称
- **门票**: 50元
- **游玩时间**: 1.5-2小时
- **特色**: 园内建筑布置精巧，奇石众多

### 3. 虎丘
- **简介**: "到苏州不游虎丘，乃憾事也"
- **门票**: 60元
- **游玩时间**: 2-3小时
- **特色**: 虎丘塔、剑池、千人石

### 4. 寒山寺
- **简介**: 因张继《枫桥夜泊》而闻名天下
- **门票**: 20元
- **游玩时间**: 1小时
- **特色**: 古钟、诗碑、夜半钟声

### 5. 平江路
- **简介**: 苏州古城保存最完整的街区
- **门票**: 免费
- **游玩时间**: 2-3小时
- **特色**: 小桥流水、古街古巷、文艺小店

### 6. 苏州博物馆
- **简介**: 贝聿铭设计，建筑本身就是艺术品
- **门票**: 免费（需预约）
- **游玩时间**: 2-3小时
- **特色**: 现代建筑与传统园林的完美结合

## 特色美食

### 苏式面
- 推荐: 同得兴、朱鸿兴、裕兴记
- 特色: 头汤面、枫镇大面、三虾面

### 苏帮菜
- 推荐: 松鹤楼、得月楼
- 特色菜: 松鼠桂鱼、响油鳝糊、清炒虾仁

### 小吃
- 哑巴生煎: 生煎包
- 潘玉麟糖粥: 糖粥、赤豆圆子
- 采芝斋: 苏式糖果、蜜饯

## 行程推荐

### 一日游
上午: 拙政园 → 苏州博物馆
下午: 平江路 → 寒山寺

### 两日游
**Day 1**: 拙政园 → 苏州博物馆 → 平江路
**Day 2**: 留园 → 虎丘 → 观前街

### 三日游
**Day 1**: 拙政园 → 苏州博物馆 → 平江路
**Day 2**: 留园 → 虎丘 → 山塘街
**Day 3**: 同里古镇 或 周庄古镇（一日往返）

## 交通指南

### 外部交通
- 高铁: 苏州站、苏州北站
- 自驾: 沪宁高速、苏嘉杭高速

### 市内交通
- 地铁: 1号线、2号线、3号线、4号线
- 公交: 票价1-2元
- 出租车: 起步价10元

## 最佳旅游时间

- **春季(3-5月)**: 百花盛开，气温宜人
- **秋季(9-11月)**: 天高气爽，桂花飘香
- **夏季**: 较热，建议室内景点为主
- **冬季**: 较冷，可体验年味

## 住宿推荐

### 高端酒店
- 苏州吴宫泛太平洋酒店
- 苏州香格里拉大酒店

### 精品民宿
- 平江路周边客栈
- 山塘街特色民宿

### 经济型
- 如家、汉庭等连锁酒店

## 购物推荐

- 观前街: 商业步行街
- 十全街: 特色小店
- 苏州中心: 大型购物中心
- 特产: 苏绣、碧螺春茶、苏式糕点

## 实用贴士

1. 园林景区建议早上开门就去，避开人流高峰
2. 苏州博物馆免费但需提前在官网预约
3. 平江路和山塘街适合傍晚散步
4. 苏式面馆建议早上品尝头汤面
5. 周末和节假日园林门票可能涨价
